## 🏁 Getting Started <a name = "getting_started"></a>
---
First, clone this template repository:
```bash
git clone https://github.com/<your-username>/<template-repo-name>.git
```

Navigate into the project folder:
```bash
cd <template-repo-name>
```

Install the dependencies:
```bash
yarn install
```

Run the development server:
```bash
yarn dev
```

To build for production:
```bash
yarn build
```

Ensure your environment satisfies all [pre-requisites](#prerequisites) before running the setup.

## 📜 Description <a name = "description"></a>
---
This is a reusable and extensible template designed to bootstrap **React** frontend applications using **Vite** and **TypeScript**. It provides modern defaults and development experience, including:

- Vite for fast build and HMR
- TypeScript support
- Pre-configured ESLint and Prettier
- Absolute import paths support
- CSS Modules setup
- Environment variable handling
- Basic folder structure for scalability

## ✅ Pre-requisites <a name = "prerequisites"></a>
